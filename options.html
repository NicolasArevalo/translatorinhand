Why does the extension need your API KEY?

I made this extension because I actually have the need of translate many words I see around. I built the extension for fun because I wanted to be able to use it for my own purposes. Now I want to publish it for free, so anyone can use it. Nontheless, the API have a limitation, as you can see if you follow the instructions, the free plan allows you to translate 500.000 characters. So if I publish it for free, the characters will run out so fast. The other option was to pay the pro Plan, but I can't afford it right now for a free extension, and I don't want to charge the users (guessing I will have). So it's better if you get your own API KEY and use it.
Notice I'm not able to see your key, it is saved locally for your own use.

Intructions

Steps:

  • Create account
  • Cancel Pro Plan
  • Get Api Key

  1. Go to Deepl, and create an account.

  2. Once in there, click on Plans and Pricing:
  3. Change to "For Developers" and sign up in free plan:
  4. Once there, fill up the form, you need to use a credit or debit card, but don't worry, it won't charge you anything.

When you create a free account, Deepl automaticly gives you a Deepl pro free trial. So you need to cancel it, I'll show you how:

  1. Click on your profile picture, and go to your account:
  2. Then scroll down until Cancel Subscription and cancel it:

Now, let's get the API KEY:

  1. So go to your profile picture, and click on your account:
  2. Change to Account:
  3. And then, scroll down and copy the API KEY
  4. That's it. Alt+T (or click on the extension logo) go to settings again and paste there your key.
